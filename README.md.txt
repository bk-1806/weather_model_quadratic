# Weather Modeling Using Quadratic Equation

This project demonstrates how to use the quadratic equation:

    T(t) = at² + bt + c

as a simplified model to predict temperature at a given time `t`.

## Versions Included

### Version 1: Hardcoded Values  
Predict temperature with pre-defined coefficients and time.

### Version 2: Keyboard Input  
Allows user to input `a`, `b`, `c`, and `t`.

### Version 3: File Input (Single Set)  
Reads `a`, `b`, `c`, and `t` from `inputs_single.txt`.

### Version 4: File Input (Multiple Sets)  
Processes multiple lines of inputs from `inputs_multiple.txt`.

## How to Run

Make sure Python is installed. Then:

```bash
python version1_hardcoded.py
python version2_keyboard_input.py
python version3_file_input_single.py
python version4_file_input_multiple.py
